# Running a workflow

Run a workflow from Nebula's web interface, or use the CLI.

To run a workflow, click on **Run Workflow**, or use the `workflow run` command:

```
./nebula-cli/nebula-cli workflow run --name <NAME_OF_WORKFLOW>
```
